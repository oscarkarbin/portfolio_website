PayMe (Chalmers group project)

Github Link

PayMe is a mobile application made natively for android. It was made with: object oriented Java, XML, Junit and JSON in android studios The purpose of the app was to solve the issue of keeping track of debts to other people. It works by creating groups of people that participate in events, a dinner for example, where someone has paid for someone else (i.e. a debt is created). An algorithm then calculates outgoing and ingoing debts for each member in the group. This debt can later on be settled through the app.
